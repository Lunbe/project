蕭婉倫_Python人工智慧與數據分析班2025年01期_主要專題報告

一、專題名稱

用十年信用卡網購數據，看見台灣消費行為的變化與年齡層趨勢

二、專題說明

  • 探討的核心問題
    • 台灣國人從 2014 年至今，網購金額與筆數的整體變化趨勢為何？
    • 哪些年份或月份出現明顯的消費高峰或轉折點？是否與政策或社會事件（如疫情）有關？
    • 各年齡層的網購消費行為是否隨時間改變？是否有特定族群快速成長？
    • 是否出現消費升級（單筆金額上升）或消費頻率增加（筆數成長）的現象？
    • 電商節日（如雙11、母親節等）是否明顯影響月度消費變化？
  • 可進行的分析方向
    • 2014 ~ 2024 年各年度消費總額與筆數變化與年增率分析
    • 觀察是否有明顯成長跳躍點（例如疫情期間）
    • 比較各年齡層消費金額與筆數年增變化，分析不同世代的行為差異
    • 分析單筆金額變化 ➜ 是否反映消費習慣轉變
    • 觀察月份與季節波動 ➜ 是否與特定促銷節慶有關
  • 專題亮點
    • 結合政策與社會事件（如疫情、電商發展）進行時間序列觀察
    • 使用 Power BI 建立互動式視覺化儀表板 ➜ 可篩選年份、月份、年齡層
    • 可視化呈現：
      • 每年 / 每月消費總額變化折線圖
      • 年齡層消費行為變化趨勢圖
      • 單筆金額與筆數的年對年比較
  • 進階分析
    • 搭配電商節慶月份清單 ➜ 分析促銷活動對消費刺激的影響
    • 若整合公開資料（如人口結構、網路/手機普及率），可進一步觀察數位落差對消費行為的潛在影響

四、資料來源

政府資料開放平台（https://data.gov.tw）

成員：蕭婉倫

分析目的與預計提供見解：

本專題延伸電商消費行為分析主題，透過十年信用卡網購數據，深入觀察台灣消費者的時間趨勢與族群變化，協助電商與品牌掌握策略重點。預期可提供以下見解：

  • 時間節點布局建議：掌握每年消費高峰月（如雙11、母親節、年底購物季），作為廣告與活動部署依據
  • 族群分眾經營建議：找出成長快速的年齡族群，協助制定分眾行銷策略
  • 消費模式趨勢建議：若發現單筆金額下滑、但筆數上升 ➜ 建議導入訂閱制、快閃組合、小額促銷策略

補充說明：若後續能整合地區 / 人口 / 數位使用相關資料，仍可進一步補強地區差異分析與數位落差導因，作為更全面的佈局參考。
